## The Moment

* The year is 2010
* The Internet is everywhere, but is not yet in every thing
  * But a rapid drop in electronics will change that
* If we put radios everywhere, will they still work?

---

## The Goal

* Big data and a smarter Internet of Things
  * Increased data should lead to smaert decisions
  * The requires putting sensors and radios on everything
* We still haven't gotten there

---

## Problems

* One problem is that we weren't ready to actually use all of that data
* But another issue was more fundamental
  * Wireless capacity isn't limitless
  * If everything has a sensor, how will we hear their updates?

## Lifetimes

* We aren't discussing smart light bulbs
  * Wireless sensors aren't wired
  * They must run on a batter or scavenge power

* **The goal:** sensors with the same lifetimes as what they are sensing
  * For example, a sensor in a road that is scraped up with the old asphalt during resurfacing
